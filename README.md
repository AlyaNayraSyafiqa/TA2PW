# Portofolio Pribadi Alya Nayra Syafiqa  
### Tugas Akhir Praktikum Pemrograman Web (JUDUL 2)

Website ini merupakan proyek portofolio pribadi yang dibuat sebagai bagian dari Tugas Akhir Praktikum Pemrograman Web.  
Aplikasi ini dibangun menggunakan HTML dan CSS tanpa bantuan framework atau JavaScript.  
Tujuan pembuatan website ini adalah untuk menampilkan profil pribadi, proyek-proyek yang telah dikerjakan, serta informasi kontak dengan tampilan yang sederhana, menarik, dan responsif.

## Fitur Utama
- Navigasi (navbar) interaktif dengan efek hover.  
- Halaman "Tentang" berisi foto dan deskripsi diri.  
- Halaman "Proyek" menampilkan beberapa project.  
- Halaman "Kontak" berisi tautan media sosial dan formulir pesan.  
- Footer berisi informasi hak cipta.  


## Teknologi yang Digunakan
- HTML : Menyusun struktur dan konten halaman web.  
- CSS : Mengatur tampilan dengan Flexbox, animasi, dan transisi.  
- Git & GitHub : Untuk version control dan publikasi project.
